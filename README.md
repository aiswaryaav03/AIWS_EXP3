## Experiment 3: Responsive Website using Bootstrap
- **Goal:** Develop a mobile-first responsive portal using the Bootstrap 5 framework.
- **Key Features:** 
  - Bootstrap 12-column Grid System (`container`, `row`, `col-*`).
  - Responsive Navigation Bar with toggle functionality.
  - Styled Bootstrap Components (Cards, Buttons, Forms).
  - Zero custom CSS; utilized Bootstrap utility classes.
